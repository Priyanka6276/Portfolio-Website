Space Fighter Game

This game is built entirely in Javascript with HTML.

About This Project

The idea behind this game was to implement what I learned so far in Javascript. This is one of the first games I created while learning Javascript.

The player can attack the alien ship or retreat which will result in a game over. When the player attacks, the alien ship also has a chance to attack. The player and the alien ship both have a chance of missing their attacks. Once the player defeats all the alien ships the game is over.

In the future I want to improve the ui/ux of the game and make it feel like a real game.
