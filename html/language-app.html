About This Project

The idea behind this web application was to allow foreign language learners to organize their vocabulary words and notes in one place.

I was inspired to create this application by my own experience of learning a language. I have a journal where I keep all my notes and a list of vocabulary and it can get filled up fast. So I wanted a place where I can organize these notes and words in one place without fear of running out of pages.

The application features user authentication to protect the users privacy. Each user can create a vocabulary word or notebook page. These words and pages can be edited or deleted.

In the future I hope to create a flashcard feature on the app and a translation feature. I also want to add a feature where users can interact with each other.
